# BABY-NAME

## Overview
This Python project suggests first names for a future child.
It is a simple and fun utility for generating naming ideas.

## Features
- Suggests baby first names
- Lightweight and easy to run
- Beginner-friendly Python project

## Requirements
- Python 3.x

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Kipre-Georges/BABY-NAME.git
   cd BABY-NAME
   ```
2. Run the script:
   ```bash
   python main.py
   ```

> If the entry file is not `main.py`, replace it with the correct filename.
